Язык C#


Язык C# был разработан компанией Microsoft и он идеально подходит для
создания программ под Windows. Компания Microsoft всячески продвигает этот
язык, обновляет и старается разрабатывать все приложения на нём. Изучение этого
языка позволит вам создавать программы под Windows и запускать их на любом
компьютере с ОС Microsoft.
