Движок Unity

Реализация мобильных игр намного удобнее при помощи игровых движков. Для
создания 2D игр под мобильные устройства стоит использовать игровой движок
Unity или GameMaker. Они позволяет создавать игры сразу под несколько
платформ (iOS, Android) и обладают большим набором функционала.
