Движок Unity

Реализация мобильных игр намного удобнее при помощи игровых движков. Для
создания 3D игр под мобильные устройства стоит использовать игровой движок
Unity. Он позволяет создавать игры сразу под несколько платформ (iOS, Android) и
обладает большим набором функционала.
