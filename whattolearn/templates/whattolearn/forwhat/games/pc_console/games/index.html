Язык C++

Для создания крупных игр со множеством элемнетов вам стоит изучать язык C++.
Язык позволяет написать свой игровой движок или же воспользоваться уже
готовыми движками, как например движок Unreal Engine 4. C++ позволяет
создавать игры класса ААА, которые считаются наиболее передовыми играми
современности.
