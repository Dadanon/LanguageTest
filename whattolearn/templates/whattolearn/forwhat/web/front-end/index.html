Язык JavaScript


Внешний вид сайта или другими словами Front-end, не состоит из лишь одного
языка программирования. Для создания внешней части сайта необходимо изучить
как минимум: HTML, CSS, SASS, Gulp, Bootstrap и JavaScript.

Набор не маленький, но это базовый набор для хорошего верстальщика. JavaScript
в свою очередь обладает уймой библиотек и фреймворков, которые расширяют
его функционал. Начните с базового комплекта и плавно передвигайтесь к
изучению React JS, Vue JS, Angular JS и Node JS. Все эти дополнительные
технологии не являются обязательными, но обладая навыками работы с ними вы
сможете найти более высокооплачиваемую работу.
