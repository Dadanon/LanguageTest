Язык PHP


PHP это наиболее популярный язык для веб программирования. По статистике его
используют в более 80% всех веб сайтов мира. Если вам необходимо создать сайт
для себя или поработать на фрилансе, то язык идеально вам подойдет

Язык обладает большим объемом встроенных функций, позволяет выполнять все
стандартные задачи, имеет большое комьюнити и легок в изучение.

Помимо языка PHP вам также потребуется: HTML, CSS, работа с базами данных SQL
и немного JavaScript.
